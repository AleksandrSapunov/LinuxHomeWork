Что такое ветвь (ветка) в терминологии GIT?
(Легковестный проэкт(ссылка на список коммитов))Это ответвление от основного проэкта для реализации своей работы либо для себя любимого
либо для чего-то ещё что бы не испортить ветку (основного проекта(master)), после завершения своей
задачи я так понимаю происходит слияние (тоесть добавление) своей работы в основу(если есть нужда).

Какой командой создается новая ветка?
git branch ...(название ветки).

Как переключиться на другую ветку?
git branch -v  (показывает ветки если не знаем)
git checkout ... (название ветки).

Как отменить изменения в раб.каталоге, которые не были добавлены в индекс и не были сохранены в репозиторий (допустим, после экспериментов)?
git checkout master - возвращение на последний коммит или вместо master (название файла) -тогда конкретный файл
если я не ошибаюсь

Каким образом можно удалить файл из очереди на коммит?
git reset ... -удаление из индекса на коммит какой-то фал или без опций(название файла) всё в каталоге

Каким образом можно удалить файл из репозитория, но оставить в раб.каталоге?
git rm --cached он станет не отслеживаемым но саму правку ещё нужно будет закомитить
git rm назв.файла "и коммит" удаление и комит  одной командой

Каким образом можно удалить файл и из репозитория и из раб.каталога?
В принцепе после предидущей команды можно просто физически его удалить из каталога
он всё равно уже не отслеживаемый через rm или клацнуть мышкой и удалить, надеюсь ты именно это имел ввиду

Как вывести на экран журнал коммитов?
git log -список коммитов текущей ветки 
git log <имя_ветки> - журнал коммитов одной из веток на выбор.

Как удалить локальный git-репозиторий текущего проекта и какие последствия у этого действия?
git rf .git - удаляет привязку к локальную привязку к гиту теперь все файлы не отслеживаемые 
и их нельзя комитить. Теперь проект становится полностью не защищённым от потери данных.
