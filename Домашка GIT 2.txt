для полной очистки индекса
git reset она очищает опционально указатель HEAD
git reset ... (... название файла) применимо к конкретному файлу 

для просмотра разницы между раб.каталогом и посл. комvитом
git diff HEAD - между рабочим каталогом и последним коммитом

для создания новой ветки и одновременного переключения в на нее
git checkout -b ... (... название ветки)

для сравнения веток между собой
git diff ... - ... вместо (... название ветки)

для загрузки ветки develop на удаленный репозиторий (GITHUB)
git push -u origin ... (... название ветки)

для удаления локальной ветки
git branch -d удаление если было слияние (-D даже без слияния)

для слияния веток
git merge ... (... название ветки) - находясь при этом в той ветке с которой совершаешь слияние

для открытия (просмотра) файла по состоянию на какой-то коммит
git show (1):(2) -(1) коммит в котором находится этот файл(2) собственно название файла без проблела после двоеточих
