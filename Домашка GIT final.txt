(1) Что делает команда git revert и пример ее использования.
git revert откатывает коммит путём создания нового я так понимаю просто его перезаписывает.
git revert (4 первых знака того коммита или все) перезаписывает этот комит путём создания нового
git revert --abort отмена последнего ревёрта 
 
(2) Какой командой можно изменить последний коммит?
git revert HEAD перезаписывает последний коммит таким же образом
git reset --hard HEAD удаляет его можно сказать рукурсивно не спрашивая 

(3) Что делает команда git remote rm origin?
Это команда используется для создания записи о новом подключении к удаленному репозиторию.

(4) В каких случаях происходит MERGE CONFLICT?
Это происходит при попытке слияния веток и файлы будут не совпадать(тоесть будут разные)
в таком случае можно сделать слияние вручную
git merge --abort можно отменить попытку слияния 

(5) Как посмотреть отличия между ветками?
git diff ... ... (... - название ветки)

(6) Какой командой удалить локальную и удаленную ветки?
git branch -d удаляет локально если было слияние (-D даже если не было слияния);
git push origin --delete ... (... - название ветки) удалить ветку удалённую;

(7) Как сделать файл неотслеживаемым (untracked)?
git rm -- cashed удаляет только из репозитория станет не отслеживаемым но саму правку ещё нужно будет закомитить
